## 🎯 Sobre o Projeto
Este projeto foi desenvolvido como requisito para a disciplina de Projetos em Sistemas. Nosso objetivo é atender a uma demanda real do(a) **Ciocolatto D'amore**, que atua no setor **Alimentício**. 

O sistema visa resolver a dificuldade de **Falta de controle de quantidade, Ausência de controle de estoque e Controle manual em planilha, dificultando análises mais detalhadas.** através de uma plataforma web rápida, responsiva e com fluxos automatizados.

## ✨ Funcionalidades
- [x] Cadastro e autenticação de usuários
- [x] Painel administrativo (Dashboard)
- [x] Automação de envio de e-mails/mensagens (via n8n)
- [ ] Geração de relatórios em PDF *(Em breve)*
